# compiladores
Repositório contendo o código do compilador desenvolvido para a disciplina de Compiladores da Poli-USP

# How to Compile
Rode o comando "make all" no diretorio em que o Makefile está presente

# How to Run
Rode o comando ./main <nome do arquivo> (no caso, o sample_program.cmm) na pasta raiz do repositorio (talvez seja necessário mudar as permissões de execução do arquivo com chmod)
